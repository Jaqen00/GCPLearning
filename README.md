# GCPLearning

GCP学习辅助工具是一款面向国家药品监督管理局高级研修学院相关 GCP 课程学习场景的桌面辅助工具。  
它用于帮助用户更顺畅地完成课程播放、状态查看、续播和下一节切换等常见操作。

> 本工具仅用于辅助课程播放与学习管理，不能替代真实学习。请以认真学习课程内容为前提合理使用，勿将其用于脱离学习目的的滥用场景。

## 项目背景

在 GCP 课程学习过程中，用户经常会遇到这些问题：

- 进入课程学习流程较繁琐
- 登录状态可能失效，需要重新确认
- 单节课程播放结束后，需要手动切换下一节
- 已有观看轨迹时，恢复进度不够准确

GCP学习辅助工具的目标，是帮助用户减少这些重复、机械的播放管理操作，让课程学习流程更清晰、更顺畅。

## 主要功能

- 自动唤起学习窗口
- 自动检测当前登录状态
- 自动进入课程并识别播放页
- 自动推进下一节课程
- 支持暂停学习、继续学习
- 支持播放页自动静音
- 支持根据观看轨迹恢复到更合理的续播位置
- 尽量保留本地登录状态，减少重复登录

## 下载方式

- [前往 Releases 页面下载](https://github.com/Jaqen00/GCPLearning/releases)

通常情况下：

- macOS 用户下载 `.dmg`
- Windows 用户下载 `.exe`

下载后按系统提示完成安装或运行即可。

## 使用方法

1. 打开 GCP学习辅助工具
2. 点击“开始学习”
3. 程序会自动唤起学习窗口
4. 如果当前未登录，请先在学习窗口中完成登录
5. 登录成功后，程序会继续自动进入课程并辅助播放

## 使用提醒

- 请始终以认真学习课程内容为前提使用本工具
- 本工具主要用于辅助播放与学习管理，不建议脱离真实学习场景使用
- 如果课程内容、登录状态或观看轨迹发生变化，请以页面实际显示为准
- 登录状态和播放相关数据仅保存在本机本地目录中

## 开发者使用指南

本项目同样面向希望继续完善、调试或二次构建的开源开发者。  
如果你希望从源代码运行或自行构建应用，可以参考下面的方式。

### 环境要求

- Node.js 20 或更高版本
- npm
- macOS / Windows 任一开发环境

### 获取源码

```bash
git clone https://github.com/Jaqen00/GCPLearning.git
cd GCPLearning
```

### 安装依赖

```bash
npm install
```

### 启动开发模式

```bash
npm run dev
```

这个模式适合日常调试与界面开发，Electron 主进程和前端界面都会一起启动。

### 预览生产版本

```bash
npm run preview
```

这个模式会先按生产方式构建，再启动 Electron 应用，适合验证接近正式发布时的运行效果。

### 构建应用资源

```bash
npm run build
```

构建完成后，应用资源会输出到 `out/` 目录。

### 本地构建安装包

如果你希望在本地直接生成安装包，也可以使用：

```bash
npm run dist:mac
```

或：

```bash
npm run dist:win
```

构建产物默认输出到：

```text
release/
```

### 说明

- 普通用户建议直接从 Releases 页面下载，不需要自行构建
- 开发者如果要参与修改，建议先使用 `npm run dev`
- 如果你计划继续扩展项目，建议在自己的分支中进行开发与测试
